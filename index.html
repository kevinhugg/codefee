O que é Javascript?

JavaScript é uma linguagem de programação de alto nível usada para adicionar interatividade às páginas web. Ela é executada no navegador do usuário, permitindo a modificação dinâmica de conteúdo, interação com elementos da página e comunicação com servidores web. Sendo constituída principalmente pela entrada de dados, processamento e saída.

Intro

As variáveis em Javascript são utilizadas para armazenar dados, podendo ser eles temporários ou permanentes. Pode-se declarar as variáveis utilizando `var`, `let`, ou `const`, atribuindo os seus valores com `=` e concatenando com o sinal de '+'. Essas variáveis podem ser strings, números, etc. Como boa prática, deve-se declarar as variáveis nas primeira linhas do código para que elas sejam as primeiras a carregar e evitar problemas, também é de grande importância dar nomes simples e faceís de lembrar.

Strings

No Javascript uma string é um tipo de dado que representa uma sequência de caracteres podendo ser letras, números, espaços e caracteres especiais. Para serem representados nós usamos as aspas duplas "", ou as aspas únicas ''. Essas aspas mostram que aquele tipo de dado não é um numeral como 1 item ou 2 itens, mesmo recebendo numerais eles se transformam em palavras quando entre aspas. Mesmo as strings sendo imutáveis elas podem ser manipuladas, que retornam com novas strings.
